## Individual Prompt Files
Each song has a dedicated prompt file in: data/cultural-prompts/

## Processing Instructions
1. Open individual prompt file (e.g., "Bà rằng bà rí-prompt.txt")
2. Copy entire prompt to Claude/GPT
3. Save JSON response to: data/cultural-context-outputs/[song-name]-context.json
4. Run merge script to integrate into lyrics files

## Batch Processing Option
For faster processing, you can combine multiple songs (5-10) into one LLM request:
- Songs 1-10: [list]
- Songs 11-20: [list]
- etc.
