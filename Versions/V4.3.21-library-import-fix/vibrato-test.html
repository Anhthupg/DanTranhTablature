Vibrato System Test - Vietnamese Dan Tranh

Vietnamese Music Rule: Vibrato applies to entire pitch classes across all octaves.

Example: Enabling F vibrato affects F4, F5, F6, etc.

How to Test:

  • Check the pitch class checkboxes below to enable vibrato
  • Vibrato starts at note center and ends when next note on same string is plucked
  • If no next note, vibrato continues for half note duration (42.5px)
  • Only used strings and used notes show vibrato
