Semantic Logic: Size differentiates, Texture indicates patterns, Outline shows modifiers

1. Core Philosophy

Semantic Hierarchy

  • Base Color: ALL notes use teal (#008080) - main and grace notes
  • Size: Differentiates note importance (main = large, grace = small)
  • Texture Fill: Shows pattern membership (dots = KPIC, lines = KRIC)
  • Outline Color: Indicates tonal modifiers (rising, falling, level)
  • Opacity/Glow: Shows selection and audio states

10. Sankey Diagram Implications

Consistent Logic in Sankey

Since all notes use the same base color:

  • KPIC-2 Sankey nodes: All teal (#008080)
  • KRIC-2 Sankey nodes: All teal (#008080)
  • Transition bands: Gradient from teal to teal (with slight variation)
  • Pattern indication: Add texture overlay on bands (dots for KPIC, lines for KRIC)
  • Size differentiation: Grace note transitions use thinner bands

13. Summary of Semantic Logic

Core Principles:

  1. One base color - All notes are teal (semantic unity)
  2. Size differentiates importance - Large = main, Small = grace
  3. Texture shows patterns - Dots = pitch, Lines = rhythm
  4. Outline indicates modifiers - Tones change border color/width
  5. Animation for states - Playing, selected, hover effects
  6. Scalable system - Works for unlimited pattern numbers
